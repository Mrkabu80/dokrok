Datenschutzerklärung

Der Schutz personenbezogener Daten ist uns wichtig. Diese statische Website ist so gestaltet, dass sie ohne Benutzerkonto und grundsätzlich ohne die aktive Eingabe personenbezogener Daten genutzt werden kann.

Hosting und Serverprotokolle

Beim Aufruf einer Website werden technisch erforderliche Verbindungsdaten verarbeitet. Dazu können IP-Adresse, Datum und Uhrzeit, aufgerufene Datei, übertragene Datenmenge, Browser, Betriebssystem und Referrer gehören. Die Verarbeitung erfolgt durch den Hostinganbieter zur sicheren Bereitstellung, Fehleranalyse und Abwehr von Missbrauch.

Kontaktaufnahme

Die Kontaktfunktion dieser Website öffnet das lokal eingerichtete E-Mail-Programm. Angaben werden nicht über einen eigenen Webserver von DOK ROK übertragen. Sobald Sie eine E-Mail senden, werden die von Ihnen mitgeteilten Daten zur Bearbeitung der Anfrage und möglicher Rückfragen verarbeitet.

Cookies und Analyse

Die bereitgestellte statische Version verwendet keine eigenen Analyse-, Marketing- oder Tracking-Cookies. Im Browser wird lediglich eine Sprachpräferenz im Local Storage gespeichert, wenn Sie die Sprache wechseln. Diese Information bleibt auf Ihrem Gerät.

Externe Links

Beim Aufruf externer Websites gelten die Datenschutzbestimmungen des jeweiligen Anbieters.

Ihre Rechte

Im Rahmen der gesetzlichen Voraussetzungen bestehen insbesondere Rechte auf Auskunft, Berichtigung, Löschung, Einschränkung der Verarbeitung, Datenübertragbarkeit und Widerspruch. Anfragen können an contact@dokrok.com gerichtet werden.
